API Documentation

GET

Initialize the database with seed data /api/initialize-db

Note: Initalize databse before making requrests or else you will run out of sql error


GET

API for statistics /api/get-statistics-data?month=[your month number] default 1
- Total sale amount of selected month
- Total number of sold items of selected month
- Total number of not sold items of selected month


GET

API for bar chart /api/get-graph-data?month=[your month number] default is 1
contains price range and the number of items in that range for the selected month regardless of the year
- 0 - 100
- 101 - 200
- 201-300
- 301-400
- 401-500
- 501 - 600
- 601-700
- 701-800
- 801-900
- 901-above


GET

API for pie chart /api/get-piechart-data?month=[your month number] default is 1
Returns categories and number of items from that category for the selected month regardless of the year.
For example :
- X category : 20 (items)
- Y category : 5 (items)
- Z category : 3 (items)


GET

API which fetches the data from all the 3 APIs mentioned above /api/get-combined-data?month=[your month number] default is 1
combines the response and sends a final response of the combined JSON
